<h1>Description</h1>
Snake is a classic arcade game that gained global popularity when Nokia included it on its phones in 1997. Developed by Taneli Armanto, its simple mechanics and addictive gameplay made it a timeless favorite and inspired many versions across different platforms.
<br>
<h1>Game Rules</h1>
1. The snake moves on pressing the 4 arrow keys of the keyboard (up, down, left, right)
2. The aim of the game is to eat the red food particles by moving towards it and grow bigger
3. Snake should stay within the boundaries of the screen and not collide with itself
