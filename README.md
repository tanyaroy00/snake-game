<h2>Description</h2>
Snake is a classic arcade game that gained global popularity when Nokia included it on its phones in 1997. Developed by Taneli Armanto, its simple mechanics and addictive gameplay made it a timeless favorite and inspired many versions across different platforms.
<br>
<h2>Game Rules</h2>
1. The snakey moves on pressing the 4 arrow keys of the keyboard (up, down, left, right)
<br>
2. The aim of the game is to eat the red food particles by moving snakey towards it and grow bigger
<br>
3. Snakey should stay within the boundaries of the screen and not collide with itself
<br>
<h2>To Run Program And Play</h2>
run: python3 snakey.py 
<br>
and enjoy:)
